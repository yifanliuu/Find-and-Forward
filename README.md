# Find & Forward (for ASAIM)

## Test

1. Migrate contracts

```
$ npm i
$ npm i -g truffle
$ truffle develop
```

```
truffle(develop)> migrate
```

-   wait for the migration to complete before proceeding

2. Run the server

```
$ cd server
$ npm i
$ npm start
```

2. Run Vue App

```
$ cd vapp
$ npm i
$ npm run serve
```

3. Browser: test
